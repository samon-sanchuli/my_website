We can easily create a facet grid that plots bikes hired by month and year.

Look at May and Jun and compare 2020 with the previous years. What’s happening? >The dataset in 2020 has a smaller kurtosis compared to previous years. The kurtosis reveals a distribution with flat tails. The flat tails indicate the small outliers in a distribution. In this case, the numbers of bike hires per month experienced less extreme cases, and the numbers are less than that compared to previous years. This can be explained by the qurantine policies introduced during Covid periods.

tfl_absolute_monthly_change1 <- bike %>% 
  filter(year>=2016) %>% 
  group_by(month) %>% 
  mutate(expected = median(bikes_hired)) %>% 
  ungroup %>% 
  group_by(month, year) %>% 
  summarise(expected = median(expected),

  mutate(excess = actual - expected)

tfl_percent_change1 <- bike %>% 
  filter(year>=2016) %>% 
  group_by(week) %>% 
  mutate(weekly_average = median(bikes_hired)) %>% 
  ungroup %>% 
  group_by(week, year) %>% 
  summarise(weekly_average = mean(weekly_average),

  mutate(percentage_change = actual_hired / weekly_average - 1)

url <- "https://fredaccount.stlouisfed.org/public/datalist/843"

# get tables that exist on the FRED website
tables <- url %>% 
  read_html() %>% 
  html_nodes(css="table")


# parse HTML tables into a dataframe called polls 
# Use purr::map() to create a list of all tables in URL
CPI_data <- map(tables, . %>% 
             html_table(fill=TRUE)%>% 
             janitor::clean_names())
CPI_series <- CPI_data[[2]] %>% # the second table on the page contains the list of all CPI components
  select(series_id)

CPI_series_vector<- as.vector(t(CPI_series)) #pull CPI components into a vector
  
CPI <- tidyquant::tq_get(CPI_series_vector, get = "economic.data", from =  "2000-01-01")

CPI <- left_join(CPI, CPI_data[[2]],
                     by = c("symbol" = "series_id"))
CPI <- CPI %>%
  group_by(title) %>% 
  mutate(year_change = price/lag(price, 12, na.rm = TRUE) - 1, date)
            
CPI <- CPI %>%         
  mutate(title = str_remove_all(title, "Consumer Price Index for All Urban Consumers: ")) %>% 
  mutate(title = str_remove_all(title, " in U.S. City Average")) %>% #remove redundant title
  group_by(date) %>%
  arrange(date, desc(year_change))
CPI_graph1 <- CPI %>% 
  filter(date >= "2016-01-01") %>% 
  ggplot(aes(x = date, y = year_change)) +
